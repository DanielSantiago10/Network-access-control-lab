# Layered Network Security Lab

## Overview
This project demonstrates the implementation of layered security controls in a network environment, including host-based firewalls, network ACLs, and service-level access control.

## Key Features
- Host-based firewall configuration for network segmentation
- Extended ACL implementation for traffic filtering
- FTP server access control using least privilege
- Multi-layered defense strategy

## Technologies Used
- Cisco IOS CLI
- Access Control Lists (ACLs)
- Firewall Configuration
- FTP Services
- Networking (TCP/IP)

## Security Concepts Demonstrated
- Network Segmentation
- Least Privilege
- Defense in Depth
- Access Control

## Project Structure
Each section of the project is organized into separate folders with configurations, explanations, and supporting screenshots.
